# 🌌 Acerca de SpotWave

**SpotWave** es un clon de Spotify desarrollado en Kotlin para Android, enfocado en ofrecer una experiencia musical moderna con reproducción online, playlists, búsqueda inteligente y una interfaz elegante inspirada en plataformas de streaming profesionales.

La aplicación permite:

- 🎵 Escuchar música online
- 🔎 Buscar artistas y canciones
- ❤️ Guardar favoritos
- 📚 Crear playlists
- 🎧 Reproducción en segundo plano
- ☁️ Streaming multimedia
- 🌙 Dark Mode moderno

El proyecto está orientado al aprendizaje y práctica de:

- Android Development
- Kotlin
- Streaming Apps
- Material Design
- APIs REST
- Multimedia Applications
- Firebase Integration

# ⚡ Instalación

## 1️⃣ Clonar repositorio

---
